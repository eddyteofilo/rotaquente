# Obsidian Eats - Digital Menu & AI Order Manager

Este projeto é uma simulação imersiva de uma operação de delivery automatizada por Inteligência Artificial. Inclui uma Landing Page profissional, simulação de WhatsApp Web, cardápio interativo e dashboard administrativo.

## 🚀 Como fazer o Deploy na Vercel

Siga os passos abaixo para colocar sua aplicação online em menos de 2 minutos:

1.  **Crie uma conta na Vercel**: Vá para [vercel.com](https://vercel.com) e conecte sua conta do GitHub/GitLab.
2.  **Importe o Projeto**: Selecione o repositório onde você subiu este código.
3.  **Configurações do Framework**:
    *   O Vercel detectará automaticamente o **Vite**.
    *   **Build Command**: `npm run build`
    *   **Output Directory**: `dist`
4.  **Instale e Deploy**: Clique em "Deploy" e aguarde a conclusão.

### 🛠️ Tecnologias Utilizadas
- **React 18**: Framework de interface.
- **Vite 5**: Ferramenta de build ultrarrápida.
- **Tailwind CSS**: Estilização moderna e responsiva.
- **Framer Motion**: Animações de alta fluidez.
- **Lucide React**: Biblioteca de ícones premium.

## 📦 Scripts Disponíveis
- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Gera a versão de produção na pasta `/dist`.
- `npm run preview`: Visualiza o build localmente.
